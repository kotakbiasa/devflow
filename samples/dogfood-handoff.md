## How it works (map)
1. `collectGitContext` → branch, base range, status, log, diff stat/diff, changed file snippets
2. `systemPrompt` + `userPrompt` → structured markdown contract
3. If key present → Anthropic Messages API; else offline skeleton
4. Write file with HTML comment metadata header
5. `track()` appends kill-metric events

Entrypoints:
- Bin: `bin/devflow.js` → `dist/cli/index.js`
- Build: `npm run build` (`tsc`)

## Risks & landmines
- **Live API path** not exercised if no `ANTHROPIC_API_KEY` in env — always smoke with `--offline` plus one manual BYOK run before public claims.
- Waitlist is **localStorage-only** until Pages + Worker; clearing browser loses signups.
- Context caps can omit critical files on huge PRs — authors should pass `--extra` notes.
- Positioning death mode: users hear “chat with repo” — copy must stay artifact-first.
- Cloudflare Pages deploy needs `CLOUDFLARE_API_TOKEN` (not set in this environment).
